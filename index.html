EF Core - NotSupportedException
Created2025-05-28|資料疆界的航圖
NotSupportedException: 描述: 當試圖使用資料庫不支援的功能時引發，或者當特定操作不適用於當前操作的提供者時引發。 例子: 嘗試使用某些特定於 SQL Server 或 SQLite 的功能，但資料庫供應者不支援這些功能。
EF Core - TimeoutException

TimeoutException: 描述: 當嘗試與資料庫進行交互時，操作超時未能完成時引發。 例子: 執行一個複雜的查詢或更新操作，但是在預定的時間內未能完成。
