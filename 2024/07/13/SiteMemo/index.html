網站的起源

Created2024-07-13|Updated2024-07-14|Others
|Post Views:

這個網站的起源，其實就是想要與這個世界做個區隔，讓我可以盡情消耗時間在精神交流上。現實中買不起房？沒關係，網路上總有我們的一席之地！

在這媒體都在討論 AI 的時代，每個人都可以擁有一個隨身的程式助理。架設一個簡單的網站對外行人而言已不再如此高不可攀。然後你會看到一個有趣的現象，有人很開心地靠 AI 為自己弄出了一個網站，與眾人分享喜悅，你以為大家會為你感到開心與驕傲嗎? No, No，更多人會開始不知道哪根筋不對，潑你一身冷水，開始笑你不自量力的肖想跟工程師搶工作，還是洗洗睡比較快

但說真的，科技的發明與創新不就是要讓更多普通人有一窺世界面貌的途徑嗎，怎變成互相輕視與角逐的工具呢 ? 難怪科技一直在進步，人們的時間反而越來越少，媒體每天為了搶時間搶眼球，喜歡下一些 類似 “大家都震驚” “大家都這樣覺得” 的標題，彷彿你一定會認同的標題，時常還會販賣焦慮，這種看到我一律滑掉，與其看這種文我不如去研究毛吉今天的大便是甚麼形狀

對我而言，能入行程式語言這項藝術全因為我有幸活在這個時代，對我來說並不是用來與人競逐攀比薪水的工具，而是能增加生活美感的東西，每一行 Code 都可以是一幅畫，也是在這世界上某個角落的設計師的想法實現
(有時候在看 Code 時腦中會浮現一些畫面，所以請不要在我對著螢幕程式自言自語顆顆笑的時候在意我)

在我眼中，程式不只是為了解決問題的工具，更是一種藝術形式。它不僅要能動，還要優雅，我很嚮往這種可以自己動手做的感覺。

我不是甚麼專家或高手，我感謝這個世界，我只是這門藝術的信徒

在這世界的一隅擁有一片小小天地讓我的精神世界任意想像與探索
