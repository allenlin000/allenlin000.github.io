AnonymousType (匿名型別)

Created2024-11-07|Updated2024-11-18|C# 舞想
|Post Views:

.NET 的匿名型別 (anonymous type) 就很有用了，它的型別資訊是由編譯器在編譯過程中動態給予的，開發人員只要用很簡單的 new 就能自由創造匿名型別。

匿名物件擁有自己的型別，無法轉型成其他型別，只有 object 除外。
匿名物件只能存在於參數，無法跨越出函數之外，除非回傳的是 object。
匿名物件宣告後，只能使用 dynamic 或 Reflection 來存取。
