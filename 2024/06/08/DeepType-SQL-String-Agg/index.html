вҳҳпёҸ дәәзҡ„дёҖз”ҹдёӯжңүдёүеӨ§йӣЈйЎҢпјҡ

  • дёӯеҚҲеҗғд»Җйәј?
  • жҳҺеӨ©иҰҒдёҚиҰҒи«ӢеҒҮпјҹ
  • жҖҺйәјжҠҠдёҖе ҶиіҮж–ҷжӢјжҲҗдёҖеҸҘи©ұпјҹ

еүҚе…©еҖӢжҲ‘йӮ„еңЁжғіпјҢз¬¬дёүеҖӢе·Із¶“жүҫеҲ°и§Јзӯ”дәҶвҖ”вҖ” STRING_AGG



д»–еёёз”ЁдҫҶдҪңзӮәж–Үеӯ—дёІжҺҘзҡ„иҒҡеҗҲеҮҪж•ёпјҢе®ҳж–№е®ҡзҫ©жҳҜйҖҷжЁЈеӯҗзҡ„ :
з”Ёж–јжҠҠ вҖңжҢҮе®ҡзҡ„ж¬„дҪҚвҖқ дёІжҲҗдёҖеҖӢ вҖңд»ҘжҢҮе®ҡеҲҶйҡ”з¬ҰеҲҶйҡ”зҡ„еӯ—з¬ҰдёІвҖқпјҢдёҚз”ЁеҜ«дёҖе ҶиӨҮйӣңзҡ„иҝҙеңҲжҲ–иҮӘиЈҪжӢјжҺҘйӮҸијҜ


жҜҸеӨ©дёҠзҸӯжү“й–Ӣе…¬еҸёзҫӨзө„пјҢе°ұеғҸжү“й–ӢдёҖеҖӢжІ’дәәж•ҙзҗҶйҒҺзҡ„иіҮж–ҷиЎЁпјҢдёҖе Ҷе»ўи©ұгҖҒиІјең–гҖҒ+1гҖҒи®ҡгҖҒдёӢеҚҲиҢ¶пјҢиӘһз„үдёҚи©ізҡ„иЁҠжҒҜиҖҢйӣ¶ж•ЈеғҸеүӣеүӣзҲҶзӮёйҒҺгҖӮжңүжҷӮзңҹзҡ„еҫҲжғіжҠҠд»–еҖ‘ STRING_AGG иө·дҫҶдёҹеҲ°д»»дҪ•зңӢдёҚиҰӢзҡ„ең°ж–№

гҖҢйӮЈеҖӢиӘ°пјҢе№«жҲ‘жҠҠйҖҷеҖӢиіҮж–ҷиЎЁиЈЎзҡ„йҖҷдәӣдёІеңЁдёҖиө·гҖҚе…үй ӯжүӢдёҠжҸ®иҲһи‘—иіҮж–ҷеғҸеңЁжӢӣйӯӮдёҖжЁЈпјҢиӘһйҹіеүӣиҗҪ

ok ж„ҹиҰәдёҚжҲіпјҢиіҮж–ҷеҜ„еҮәеҺ»и¶•еҝ«дёӢзҸӯ!

WITHIN GROUP жҙҫдёҠз”Ёе ҙпјҢжңүй»һеғҸ Window Function

иҖҒй—Ҷ : гҖҢж¬ё..жңүдёҖд»¶дәӢиҰҒй ҶдҫҝдёҖдёӢгҖҚ

(е–”? й Ҷдҫҝд»Җйәј? зҷјзҚҺйҮ‘зөҰжҲ‘е—Һ?)

иҖҒй—Ҷжү“й–ӢеҸҰдёҖеҖӢиҰ–зӘ—пјҢз§ҖеҮәеҸҰеӨ–дёҖзө„иіҮж–ҷ

иҖҒй—Ҷ : гҖҢдҪ зңӢвҖҰ йҖҷдәӣз°ЎиЁҠзҷјеҮәеҺ»зҡ„иҷҹзўјйғҪжҳҜе“Әдәӣдәәе•Ҡ?гҖҚ

е·ҘдҪңдёӯпјҢжҲ‘еҖ‘зёҪжҳҜеңЁжҺЎеқ‘пјҢдёҚеҰӮиӘӘпјҢе·ҘдҪңжң¬иә«е°ұжҳҜдёҖеҖӢеқ‘пјҢе·ҘдҪңжҲ‘еҖ‘еҜҰеңЁйҒҝдёҚжҺүпјҢдҪҶ SQL зҡ„еқ‘йӮ„жҳҜиғҪе°‘иё©дёҖй»һжҳҜдёҖй»һ


вҳ‘пёҸ з©әеҖјиҷ•зҗҶ

еҰӮжһңж¬„дҪҚеҸҜиғҪжңғжҳҜ NULLпјҢйӮЈйҖҷдәӣеҖјжңғиў«иҮӘеӢ•еҝҪз•ҘгҖӮжғіжҺ§еҲ¶дёҖдёӢпјҢеҸҜд»ҘйҖҷжЁЈеҒҡ

вҳ‘пёҸ й•·еәҰйҷҗеҲ¶

жӢјжҺҘиө·дҫҶзҡ„еӯ—дёІеҸҜиғҪи¶…йҒҺж¬„дҪҚжҲ–и®Ҡж•ёжңҖеӨ§е®№йҮҸпјҢе»әиӯ°еҠ дёҠ LEN() жӘўжҹҘпјҢжҲ–дҫқйңҖжұӮжӢҶжү№иҷ•зҗҶпјҢйҒҝе…ҚйҒҺй•·зҲҶжҺү

вҳ‘пёҸ жҮүз”ЁеҲ°еӯҗжҹҘи©ўжҲ– CTE

еҸҜд»ҘжҠҠ STRING_AGG еҢ…йҖІ CTE жҲ–еӯҗжҹҘи©ўиЈЎеҒҡжӣҙйҖІдёҖжӯҘйӮҸијҜиҷ•зҗҶгҖӮдҫӢеҰӮпјҢеңЁдёҖеҖӢз”ўе“ҒйҠ·е”®иіҮж–ҷиЎЁдёӯпјҢе°ҮжҜҸеҖӢең°еҚҖзҡ„йҠ·е”®е“ЎеҗҚеӯ—дёІжҺҘзӮәдёҖеҖӢеӯ—дёІпјҢз„¶еҫҢеҶҚеҲҶжҲҗдёІдёҠең°еҚҖиіҮиЁҠ

     еҰӮжһңдәәз”ҹд№ҹиғҪз”Ё STRING_AGG жҠҠеҝ«жЁӮзҡ„иЁҳжҶ¶дёІиө·дҫҶ
