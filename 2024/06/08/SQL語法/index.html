STRING_AGG

Created2024-06-08|Updated2024-11-19|SQL
|Post Views:

STRING_AGG 是一個在 SQL Server 中使用的聚合函數，用於將某一欄位中的多個值連接成一個以指定分隔符分隔的字符串

舉個例子

我們有一個資料表，儲存了不同訂單中包含的商品，我們希望取得每個訂單中所有商品的名稱，並將這些商品名稱連接成一個以逗號（或其他分隔符）分隔的字符串，以便顯示或進行其他操作。
