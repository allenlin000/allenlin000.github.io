GroupBy探索 (一.Selector)

Created2024-03-17|Updated2024-06-16|LINQ
|Post Views:

MS Doc 名詞解析(以音樂曲目為例)

keySelector : 將歌曲按照什麼標準分組，例如按照曲風（搖滾、爵士、古典）、調性（大調、小調）或演出者。
comparer ： 決定key是否真的相同的標準，例如 : 我們使用歌手的名字做歌曲的分組,然而,歌手資料清單中，繁體簡體是否可以忽略?
elementSelector : 選擇在展示歌曲細節，匹方說我們只列出曲名和作曲者，不會把歌詞一起列出來。
resultSelector : 格式化最終的分群歌曲清單，列出每個分群（如每種曲風）下的歌曲列表，每個分組的歌曲可能有各自的數量或者特色介紹。

MS Doc 範例

需求: 以寵物的年齡做分群(捨棄小數), 並希望輸出後的結果包含 : 年齡,數量,最老,最年輕

此處會使用到resultSelector ,他的角色就是定義我們想要怎麼輸出我們分組後的結果!

需求: 以寵物的年齡做分群(捨棄小數), 並希望輸出後的分群結果只有名字

此處會使用到elementSelector ,他的角色就是定義我們想要輸出哪些屬性在各群中!
