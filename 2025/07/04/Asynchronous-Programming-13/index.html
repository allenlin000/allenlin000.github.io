Asynchronous Programming - 第十一章:Task.WhenAll

Created2025-07-04|Updated2025-09-11|未来よりの返歌
|Post Views:

你叫三個朋友(小弟)幫你去不同的店買東西(跑腿)：

小明去買雞蛋
小華去買牛奶
小美去買麵包

你不會等小明回來才讓小華去，你會同時叫他們出門。然後你說：「等你們全部都回來，我們再一起做早餐！」這就是 Task.WhenAll ，等全部人回來，再開始做接下來的事。
它的本質是把很多 Task 包裝成「一個新的 Task」，這個新 Task 會在 所有子 Task 都完成後才完成。(to execute multiple asynchronous tasks concurrently，and wait for all of them to complete)
