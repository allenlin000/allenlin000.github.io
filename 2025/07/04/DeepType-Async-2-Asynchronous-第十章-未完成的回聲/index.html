Asynchronous Programming - уггтЇЂуФа:Тюфт«їТѕљуџётЏъУЂ▓

у░АС╗І.NET 4.0 уџётцџтиЦтЪиУАїтѕЕтЎе РђЊTask

C# тГИу┐њуГєУеў№╝џтцџтЪиУАїуињ (6) - TPL

тюеТ»ЈСИђтђІС╗╗тІЎУбФТ┤ЙжЂБтЄ║тј╗уџётцюУБА№╝їуИйТюЅСИђС║ЏУЂ▓жЪ│№╝їуёАТ│ЋтЇ│тѕ╗ТГИСЙєсђѓт«ЃтђЉтюет╣│УАїуџётЪиУАїуињУБАС║цжї»уЕ┐УАї№╝їТюЅуџёТіхжЂћС║єухѓж╗ъ№╝їУ┐ћтЏъСИђУЂ▓У╝Ћт┐Фуџёт«їТѕљ№╝ЏТюЅуџётюеуГЅтЙЁСИГУЁљУЮЋ№╝їТѕљуѓ║уёАУЂ▓уџёжў╗тАъ№╝ЏТюЅуџётЇіжђћТІІСИІУфЊУеђ№╝їТХѕтц▒тюетЈќТХѕуџёУЇњжЄјУБАсђѓСйаС╗Цуѓ║уеІт╝Јуб╝т»ФтЦйС║є№╝їТхЂтљЉти▓уёХТўјТюЌ№╝їтЇ╗СИЇуЪЦТ»ЈтђІ Task жЃйтЃЈТхЂТхфуџёС┐АС╗Х№╝їт░ЂтГўУЉЌтЈ»УЃй№╝їС╣ЪтЪІУЌЈУЉЌТюфуФЪ

TPL№╝їСИЇтЈфТў»СИђтЦЌт╣│УАїуџётиЦтЁи№╝їт«ЃТў»СИђта┤жЌюТќ╝ уГЅтЙЁУѕЄтЉйжЂІ уџёт»джЕЌ№╝їТ»ЈСИђтђІ WaitAllсђЂWhenAnyсђЂResult№╝їжЃйТў»жќІтЋЪтѕєТћ»УѕЄухљт▒ђуџётњњУфъсђѓуЋХухљТъюТюфТГИ№╝їтЏъУЂ▓С╗ЇтюесђѓтюетЪиУАїуињуџёТи▒УЎЋ№╝їТюЅС║ЏС╗╗тІЎ№╝їухѓт░Єт«їТѕљ№╝ЏТюЅС║ЏтЏъУЂ▓№╝їТ░ИжЂаТюфт«ї

тЁѕућеТюђу░Атќ«уџё Task.Run т╗║уФІСИђтђІС╗╗тІЎ№╝їуюІуюІт«ЃУиЪСИђУѕгуеІт╝Јуб╝уџётЪиУАїжаєт║ЈТюЅС╗ђж║╝СИЇСИђТеБсђѓ

тЪиУАїухљТъю

уѓ║С╗ђж║╝тЈфтЇ░тЄ║ Main Thread!!!!!№╝Ъ

жђЎТў»тЏауѓ║ Task жаљУеГТў» жЮътљїТГЦ№╝ѕAsynchronous№╝Ѕ тЪиУАїуџё№╝їСйатЈФС╗ќтј╗тЂџС║І№╝ѕжќІУЃїТЎ»тиЦСйю№╝ЅтЙї№╝їСИ╗уеІт╝ЈСИЇТюЃтЂюСИІСЙєуГЅС╗ќтЂџт«їсђѓСИ╗уеІт╝ЈтЪиУАїуињу╣╝у║їтЙђСИІУиЉ№╝їуЏ┤ТјЦТіі Main Thread!!!!! тЇ░тЄ║СЙєсђѓУђїУЃїТЎ»тЪиУАїуџё Task жюђУдЂСИђж╗ъТЎѓжќЊ№╝ѕThread.Sleep(1000)№╝Ѕ№╝їухљТъюСЙєСИЇтЈітЇ░№╝їСИ╗уеІт╝Јт░▒УиЉт«їжЌюжќЅС║є

Task.Run уџёТюгУ│фТў»сђїТіітиЦСйюСИЪухдтѕЦуџётЪиУАїуињтј╗тЂџ№╝їУЄфти▒СИЇуГЅухљТъют░▒у╣╝у║їтЙђСИІУиЉсђЇ№╝їТЅђС╗ЦтдѓТъюСИ╗уеІт╝ЈтЁѕухљТЮЪ№╝їУЃїТЎ»С╗╗тІЎТа╣ТюгСЙєСИЇтЈіт«їТѕљ

  1. Main() тЉ╝тЈФ SimpleTask()№╝їуЏ«тЅЇжѓётюесђїСИ╗тЪиУАїуињ№╝ѕMain Thread№╝ЅсђЇСИіУиЉ
  2. Task.Run(Рђд) УбФтЉ╝тЈФ№╝їCLR Тіі lambda тЁДт«╣СИЪтѕ░ Thread Pool№╝їжђЎСИђТГЦсђїтЈфТў»тюеТјњуеІС╗╗тІЎсђЇ№╝їСИЇТў»тюетЪиУАїтЁДт«╣
  3. Task.Run уФІтѕ╗тЏътѓ│№╝їСИ╗тЪиУАїуињт«їтЁеТ▓њТюЅуГЅ№╝їTask жѓётюеУЃїТЎ»Т║ќтѓЎТѕќтЅЏжќІтДІУиЉ
  4. СИ╗тЪиУАїуињуФІтѕ╗тЪиУАїСИдУ╝ИтЄ║ Console.WriteLine(РђюMain Thread!!!!!РђЮ)
  5. SimpleTask() ухљТЮЪ Рєњ Main() ухљТЮЪ№╝їConsole ТЄЅућеуеІт╝ЈуЏ┤ТјЦжЌюжќЅ
  6. УЃїТЎ» Task жѓётюе Sleep№╝їт░▒УбФт╝итѕХухѓТГб

СИђтђІТќ╣Т│ЋТў» Main тцџтіаСИђтђІ Console.Read() Тѕќ Console.ReadLine()№╝їУ«ЊСИ╗тЪиУАїуињтюеухљТЮЪтЅЇтЇАСйЈ№╝їуГЅСйауюІт«їУ╝ИтЄ║

РюЁ тЪиУАїухљТъю

тЏауѓ║тцџС║є Console.Read()№╝їуеІт╝ЈСИЇТюЃждгСИіухљТЮЪ№╝їТЅђС╗ЦуГЅтѕ░УЃїТЎ»С╗╗тІЎУиЉт«їтЙї№╝їС╣ЪТюЃуюІтѕ░ Try Task!!!!!!!!сђѓ

уЈЙтюеТѕЉтђЉУЕдУЕдуюІтљїТеБуџёТдѓт┐х№╝їжќІтЁЕтђІ Task№╝їуёХтЙїуб║С┐ЮтЁЕтђІжЃйтЂџт«їтЙїтєЇу╣╝у║їтЪиУАїСИ╗ТхЂуеІ

уюІтѕ░СИіСИђтђІСЙІтГљ№╝їТѕЉтђЉуЪЦжЂЊ Task жќІтДІтЪиУАїТѕЉтђЉт░▒СИЇТюЃтЂютюежѓБжѓіуГЅС╗ќт«їТѕљ№╝їТюЃу╣╝у║їУиЉСИ╗ТхЂуеІ№╝їтЏаТГцжђЎжѓіТѕЉтђЉуће WaitAll жў╗тАъ№╝їжаљТюЪСИі№╝їуЏ┤тѕ░ task1сђЂtask2 т«їТѕљтЅЇ№╝їMain Thread СИЇТюЃтєњтЄ║СЙє

РюЁ тЪиУАїухљТъю

уЋХСйатљїТЎѓтЋЪтІЋтцџтђІ Task ТЎѓ№╝їСИ╗тЪиУАїуињСИђТеБСИЇТюЃуГЅт«ЃтђЉсђѓTask.WaitAll Тў»СИђтђІсђїтљїТГЦжў╗тАъ№╝ѕBlocking№╝ЅсђЇуџёТќ╣Т│Ћ№╝їТюЃтЇАтюежѓБУБА№╝їуЏ┤тѕ░СйатѕЌтЄ║уџёТЅђТюЅ Task жЃйтЪиУАїт«їТЅЇтЙђСИІУиЉ

РюЁ тЪиУАїухљТъю

taskResult Тў»СИђтђІ Task тЪиУАїуІђТ│Ђ№╝їт«ЃУБАжЮбтїЁС║єСИђтђІсђїжѓёТ▓њтЂџтЦйуџёухљТъюсђЇсђѓуЋХСйатЉ╝тЈФ .Result ТЎѓ№╝їуеІт╝ЈТюЃ жў╗тАъ№╝ѕBlocking№╝Ѕ№╝їтдѓТъюухљТъюжѓёТ▓њу«ЌтЦй№╝їт░▒ТюЃтюежѓБУБАуГЅсђѓуГЅтѕ░ухљТъюу«ЌтЦйС║є№╝їт░▒Тіітђ╝тЏътѓ│ухдСйа

  • Task.WhenAny Тђјж║╝тЂџтцџС╗╗тІЎуФХУ│й№╝ѕУф░тЁѕт«їТѕљ№╝Ѕ
  • CancellationToken Тђјж║╝ТІІтЄ║тЈќТХѕСЙІтцќ
  • try/catch/finally Тђјж║╝ТГБуб║тѕєУЙе CompletedсђЂCanceledсђЂFaulted СИЅуе«уІђТЁІ
  • Тђјж║╝ТіітљїТГЦУ╝ИтЁЦ№╝ѕConsole.ReadKey№╝ЅтїЁТѕљ Task УиЪ Delay Т»ћУ│й

1№ИЈРЃБ тЁѕТ║ќтѓЎтЦйтЈќТХѕтіЪУЃй

2№ИЈРЃБ жќІСИђтђІ Task№╝їУБАжЮбжѓіуГЅУ╝ИтЁЦжѓіУеѕТЎѓ

уће Console.ReadKey() У«ЊСй┐ућеУђЁТїЅ A/B/C тЁХСИГСИђтђІТїЅжЇхсђѓТ»ЈуДњТфбТЪЦСИђТгА№╝їуће Task.WhenAny уб║УфЇСй┐ућеУђЁУ╝ИтЁЦ or ТЎѓжќЊтѕ░тЊфСИђтђІтЁѕ

3№ИЈРЃБ тдѓТъюУ╝ИтЁЦС║є№╝їт░▒СЙЮУ╝ИтЁЦтЪиУАїСИЇтљїтѕєТћ»

  • A№╝џТГБтИИт«їТѕљ№╝їтЏътѓ│ РђюOKРђЮ
  • B№╝џТІІСЙІтцќ№╝їТеАТЊгуЋ░тИИуІђТ│Ђ
  • C№╝џтЉ╝тЈФтЈќТХѕ№╝їСИЪтЄ║ OperationCanceledException
  • тЁХС╗ќ№╝џтЏътѓ│ Unknown

4№ИЈРЃБ 5 уДњжЃйТ▓њУ╝ИтЁЦт░▒тЏътѓ│ РђюNo Input!РђЮ

жђЎТЎѓ Task С╣Ъу«ЌТў»ТГБтИИт«їТѕљ

5№ИЈРЃБ СИ╗ТхЂуеІ try/catch/finally тѕєтѕЦУЎЋуљєСИЅуе«тЈ»УЃйухљТъю

  • OperationCanceledException Рєњ Canceled
  • тЁХС╗ќ Exception Рєњ Faulted
  • ТГБтИИ Рєњ Completed

т«їТЋ┤т»дСйю

Task Т▓њТюЅсђїуІђТЁІУеГт«џтЎесђЇ№╝їтЈфТюЅсђїухљТъюТјеУФќтЎесђЇ№╝їCLR тЈфТў»Та╣ТЊџСйажЏбжќІТќ╣Т│ЋуџёТќ╣т╝Ј№╝їУЄфтІЋт╣ФСйатѕєжАъсђѓтЈфУдЂСйауюІтѕ░СИђтђІ Task Тў» Completed№╝їжѓБС╗БУАесђїУБАжЮбуџёуеІт╝Јуб╝уюЪуџёТ▓њТюЅуѓИТјЅсђЇсђѓжђЎТў» async/await УЃйУбФС┐АС╗╗уџёТа╣ТюгтјЪтЏа№╝їTask уџёуІђТЁІТў»ућ▒сђїСйаТђјж║╝ухљТЮЪжѓБТ«хуеІт╝Јуб╝сђЇТ▒║т«џуџё!

­Ъћх 1. С╗ђж║╝жЃйСИЇУ╝ИтЁЦ Рєњ ТГБтИИухљТЮЪ

Console.ReadKey() Т▓њТюЅУ╝ИтЁЦ№╝їdelayTask Т»ЈТгАжЃйтЁѕт«їТѕљ
5 ТгАУ┐┤тюѕтЙї№╝їуЏ┤ТјЦтЏътѓ│ РђюNo Input!РђЮ№╝їС╗БУАеС╗╗тІЎТў»ТГБтИИт«їТѕљуџё№╝їуІђТЁІТў» Completed

­ЪЪб 2. 5 уДњтЁДУ╝ИтЁЦ A Рєњ ТГБтИИт«їТѕљ

Console.ReadKey() У╝ИтЁЦ A тЙї№╝їinputTask тЁѕт«їТѕљсђѓTask уЏ┤ТјЦУх░тѕ░ return РђюOKРђЮсђѓуІђТЁІТў» Completedсђѓ

­Ъћ┤ 3. 5 уДњтЁДУ╝ИтЁЦ B Рєњ уЎ╝ућЪСЙІтцќ

B ТюЃтЪиУАї throw new ApplicationExceptionсђѓжђЎТюЃУ«Њ Task У«іТѕљ Faulted уІђТЁІ№╝ѕУБАжЮбТюЅ Exception№╝Ѕсђѓжђ▓тЁЦ catch (Exception) тЇђтАі№╝їтЇ░тЄ║жї»УфцУеіТЂ»сђѓ

­ЪЪА 4. 5 уДњтЁДУ╝ИтЁЦ C Рєњ тЈќТХѕ

У╝ИтЁЦ C ТЎѓ№╝їтЉ╝тЈФ cts.Cancel() СИдСИћ ThrowIfCancellationRequested()сђѓжђЎТюЃТІІтЄ║ OperationCanceledExceptionсђѓTask У«іТѕљ Canceled уІђТЁІсђѓжђ▓тЁЦ catch (OperationCanceledException)
