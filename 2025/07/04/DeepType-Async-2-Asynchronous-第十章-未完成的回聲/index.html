Asynchronous Programming - з¬¬еҚҒз« :жңӘе®ҢжҲҗзҡ„еӣһиҒІ

Created2025-07-04|Updated2026-03-31|жңӘжқҘгӮҲгӮҠгҒ®иҝ”жӯҢ
|Post Views:

з°Ўд»Ӣ.NET 4.0 зҡ„еӨҡе·Ҙеҹ·иЎҢеҲ©еҷЁ вҖ“Task

C# еӯёзҝ’зӯҶиЁҳпјҡеӨҡеҹ·иЎҢз·’ (6) - TPL

еңЁжҜҸдёҖеҖӢд»»еӢҷиў«жҙҫйҒЈеҮәеҺ»зҡ„еӨңиЈЎпјҢзёҪжңүдёҖдәӣиҒІйҹіпјҢз„Ўжі•еҚіеҲ»жӯёдҫҶгҖӮе®ғеҖ‘еңЁе№іиЎҢзҡ„еҹ·иЎҢз·’иЈЎдәӨйҢҜз©ҝиЎҢпјҢжңүзҡ„жҠөйҒ”дәҶзөӮй»һпјҢиҝ”еӣһдёҖиҒІиј•еҝ«зҡ„е®ҢжҲҗпјӣжңүзҡ„еңЁзӯүеҫ…дёӯи…җиқ•пјҢжҲҗзӮәз„ЎиҒІзҡ„йҳ»еЎһпјӣжңүзҡ„еҚҠйҖ”жӢӢдёӢиӘ“иЁҖпјҢж¶ҲеӨұеңЁеҸ–ж¶Ҳзҡ„иҚ’йҮҺиЈЎгҖӮдҪ д»ҘзӮәзЁӢејҸзўјеҜ«еҘҪдәҶпјҢжөҒеҗ‘е·Із„¶жҳҺжң—пјҢеҚ»дёҚзҹҘжҜҸеҖӢ Task йғҪеғҸжөҒжөӘзҡ„дҝЎд»¶пјҢе°Ғеӯҳи‘—еҸҜиғҪпјҢд№ҹеҹӢи—Ҹи‘—жңӘз«ҹ

TPLпјҢдёҚеҸӘжҳҜдёҖеҘ—е№іиЎҢзҡ„е·Ҙе…·пјҢе®ғжҳҜдёҖе ҙй—ңж–ј зӯүеҫ…иҲҮе‘ҪйҒӢ зҡ„еҜҰй©—пјҢжҜҸдёҖеҖӢ WaitAllгҖҒWhenAnyгҖҒResultпјҢйғҪжҳҜй–Ӣе•ҹеҲҶж”ҜиҲҮзөҗеұҖзҡ„е’’иӘһгҖӮз•¶зөҗжһңжңӘжӯёпјҢеӣһиҒІд»ҚеңЁгҖӮеңЁеҹ·иЎҢз·’зҡ„ж·ұиҷ•пјҢжңүдәӣд»»еӢҷпјҢзөӮе°Үе®ҢжҲҗпјӣжңүдәӣеӣһиҒІпјҢж°ёйҒ жңӘе®Ң

е…Ҳз”ЁжңҖз°Ўе–®зҡ„ Task.Run е»әз«ӢдёҖеҖӢд»»еӢҷпјҢзңӢзңӢе®ғи·ҹдёҖиҲ¬зЁӢејҸзўјзҡ„еҹ·иЎҢй ҶеәҸжңүд»ҖйәјдёҚдёҖжЁЈгҖӮ

еҹ·иЎҢзөҗжһң

зӮәд»ҖйәјеҸӘеҚ°еҮә Main Thread!!!!!пјҹ

йҖҷжҳҜеӣ зӮә Task й җиЁӯжҳҜ йқһеҗҢжӯҘпјҲAsynchronousпјү еҹ·иЎҢзҡ„пјҢдҪ еҸ«д»–еҺ»еҒҡдәӢпјҲй–ӢиғҢжҷҜе·ҘдҪңпјүеҫҢпјҢдё»зЁӢејҸдёҚжңғеҒңдёӢдҫҶзӯүд»–еҒҡе®ҢгҖӮдё»зЁӢејҸеҹ·иЎҢз·’з№јзәҢеҫҖдёӢи·‘пјҢзӣҙжҺҘжҠҠ Main Thread!!!!! еҚ°еҮәдҫҶгҖӮиҖҢиғҢжҷҜеҹ·иЎҢзҡ„ Task йңҖиҰҒдёҖй»һжҷӮй–“пјҲThread.Sleep(1000)пјүпјҢзөҗжһңдҫҶдёҚеҸҠеҚ°пјҢдё»зЁӢејҸе°ұи·‘е®Ңй—ңй–үдәҶ

Task.Run зҡ„жң¬иіӘжҳҜгҖҢжҠҠе·ҘдҪңдёҹзөҰеҲҘзҡ„еҹ·иЎҢз·’еҺ»еҒҡпјҢиҮӘе·ұдёҚзӯүзөҗжһңе°ұз№јзәҢеҫҖдёӢи·‘гҖҚпјҢжүҖд»ҘеҰӮжһңдё»зЁӢејҸе…ҲзөҗжқҹпјҢиғҢжҷҜд»»еӢҷж №жң¬дҫҶдёҚеҸҠе®ҢжҲҗ

  1. Main() е‘јеҸ« SimpleTask()пјҢзӣ®еүҚйӮ„еңЁгҖҢдё»еҹ·иЎҢз·’пјҲMain ThreadпјүгҖҚдёҠи·‘
  2. Task.Run(вҖҰ) иў«е‘јеҸ«пјҢCLR жҠҠ lambda е…§е®№дёҹеҲ° Thread PoolпјҢйҖҷдёҖжӯҘгҖҢеҸӘжҳҜеңЁжҺ’зЁӢд»»еӢҷгҖҚпјҢдёҚжҳҜеңЁеҹ·иЎҢе…§е®№
  3. Task.Run з«ӢеҲ»еӣһеӮіпјҢдё»еҹ·иЎҢз·’е®Ңе…ЁжІ’жңүзӯүпјҢTask йӮ„еңЁиғҢжҷҜжә–еӮҷжҲ–еүӣй–Ӣе§Ӣи·‘
  4. дё»еҹ·иЎҢз·’з«ӢеҲ»еҹ·иЎҢдёҰијёеҮә Console.WriteLine(вҖңMain Thread!!!!!вҖқ)
  5. SimpleTask() зөҗжқҹ вҶ’ Main() зөҗжқҹпјҢConsole жҮүз”ЁзЁӢејҸзӣҙжҺҘй—ңй–ү
  6. иғҢжҷҜ Task йӮ„еңЁ SleepпјҢе°ұиў«еј·еҲ¶зөӮжӯў

дёҖеҖӢж–№жі•жҳҜ Main еӨҡеҠ дёҖеҖӢ Console.Read() жҲ– Console.ReadLine()пјҢи®“дё»еҹ·иЎҢз·’еңЁзөҗжқҹеүҚеҚЎдҪҸпјҢзӯүдҪ зңӢе®ҢијёеҮә

вң… еҹ·иЎҢзөҗжһң

еӣ зӮәеӨҡдәҶ Console.Read()пјҢзЁӢејҸдёҚжңғйҰ¬дёҠзөҗжқҹпјҢжүҖд»ҘзӯүеҲ°иғҢжҷҜд»»еӢҷи·‘е®ҢеҫҢпјҢд№ҹжңғзңӢеҲ° Try Task!!!!!!!!гҖӮ

зҸҫеңЁжҲ‘еҖ‘и©Ұи©ҰзңӢеҗҢжЁЈзҡ„жҰӮеҝөпјҢй–Ӣе…©еҖӢ TaskпјҢз„¶еҫҢзўәдҝқе…©еҖӢйғҪеҒҡе®ҢеҫҢеҶҚз№јзәҢеҹ·иЎҢдё»жөҒзЁӢ

зңӢеҲ°дёҠдёҖеҖӢдҫӢеӯҗпјҢжҲ‘еҖ‘зҹҘйҒ“ Task й–Ӣе§Ӣеҹ·иЎҢжҲ‘еҖ‘е°ұдёҚжңғеҒңеңЁйӮЈйӮҠзӯүд»–е®ҢжҲҗпјҢжңғз№јзәҢи·‘дё»жөҒзЁӢпјҢеӣ жӯӨйҖҷйӮҠжҲ‘еҖ‘з”Ё WaitAll йҳ»еЎһпјҢй җжңҹдёҠпјҢзӣҙеҲ° task1гҖҒtask2 е®ҢжҲҗеүҚпјҢMain Thread дёҚжңғеҶ’еҮәдҫҶ

вң… еҹ·иЎҢзөҗжһң

з•¶дҪ еҗҢжҷӮе•ҹеӢ•еӨҡеҖӢ Task жҷӮпјҢдё»еҹ·иЎҢз·’дёҖжЁЈдёҚжңғзӯүе®ғеҖ‘гҖӮTask.WaitAll жҳҜдёҖеҖӢгҖҢеҗҢжӯҘйҳ»еЎһпјҲBlockingпјүгҖҚзҡ„ж–№жі•пјҢжңғеҚЎеңЁйӮЈиЈЎпјҢзӣҙеҲ°дҪ еҲ—еҮәзҡ„жүҖжңү Task йғҪеҹ·иЎҢе®ҢжүҚеҫҖдёӢи·‘

вң… еҹ·иЎҢзөҗжһң

taskResult жҳҜдёҖеҖӢ Task еҹ·иЎҢзӢҖжіҒпјҢе®ғиЈЎйқўеҢ…дәҶдёҖеҖӢгҖҢйӮ„жІ’еҒҡеҘҪзҡ„зөҗжһңгҖҚгҖӮз•¶дҪ е‘јеҸ« .Result жҷӮпјҢзЁӢејҸжңғ йҳ»еЎһпјҲBlockingпјүпјҢеҰӮжһңзөҗжһңйӮ„жІ’з®—еҘҪпјҢе°ұжңғеңЁйӮЈиЈЎзӯүгҖӮзӯүеҲ°зөҗжһңз®—еҘҪдәҶпјҢе°ұжҠҠеҖјеӣһеӮізөҰдҪ 

  • Task.WhenAny жҖҺйәјеҒҡеӨҡд»»еӢҷз«¶иіҪпјҲиӘ°е…Ҳе®ҢжҲҗпјү
  • CancellationToken жҖҺйәјжӢӢеҮәеҸ–ж¶ҲдҫӢеӨ–
  • try/catch/finally жҖҺйәјжӯЈзўәеҲҶиҫЁ CompletedгҖҒCanceledгҖҒFaulted дёүзЁ®зӢҖж…Ӣ
  • жҖҺйәјжҠҠеҗҢжӯҘијёе…ҘпјҲConsole.ReadKeyпјүеҢ…жҲҗ Task и·ҹ Delay жҜ”иіҪ

1пёҸвғЈ е…Ҳжә–еӮҷеҘҪеҸ–ж¶ҲеҠҹиғҪ

2пёҸвғЈ й–ӢдёҖеҖӢ TaskпјҢиЈЎйқўйӮҠзӯүијёе…ҘйӮҠиЁҲжҷӮ

з”Ё Console.ReadKey() и®“дҪҝз”ЁиҖ…жҢү A/B/C е…¶дёӯдёҖеҖӢжҢүйҚөгҖӮжҜҸз§’жӘўжҹҘдёҖж¬ЎпјҢз”Ё Task.WhenAny зўәиӘҚдҪҝз”ЁиҖ…ијёе…Ҙ or жҷӮй–“еҲ°е“ӘдёҖеҖӢе…Ҳ

3пёҸвғЈ еҰӮжһңијёе…ҘдәҶпјҢе°ұдҫқијёе…Ҙеҹ·иЎҢдёҚеҗҢеҲҶж”Ҝ

  • AпјҡжӯЈеёёе®ҢжҲҗпјҢеӣһеӮі вҖңOKвҖқ
  • BпјҡжӢӢдҫӢеӨ–пјҢжЁЎж“¬з•°еёёзӢҖжіҒ
  • Cпјҡе‘јеҸ«еҸ–ж¶ҲпјҢдёҹеҮә OperationCanceledException
  • е…¶д»–пјҡеӣһеӮі Unknown

4пёҸвғЈ 5 з§’йғҪжІ’ијёе…Ҙе°ұеӣһеӮі вҖңNo Input!вҖқ

йҖҷжҷӮ Task д№ҹз®—жҳҜжӯЈеёёе®ҢжҲҗ

5пёҸвғЈ дё»жөҒзЁӢ try/catch/finally еҲҶеҲҘиҷ•зҗҶдёүзЁ®еҸҜиғҪзөҗжһң

  • OperationCanceledException вҶ’ Canceled
  • е…¶д»– Exception вҶ’ Faulted
  • жӯЈеёё вҶ’ Completed

е®Ңж•ҙеҜҰдҪң

Task жІ’жңүгҖҢзӢҖж…ӢиЁӯе®ҡеҷЁгҖҚпјҢеҸӘжңүгҖҢзөҗжһңжҺЁи«–еҷЁгҖҚпјҢCLR еҸӘжҳҜж №ж“ҡдҪ йӣўй–Ӣж–№жі•зҡ„ж–№ејҸпјҢиҮӘеӢ•е№«дҪ еҲҶйЎһгҖӮеҸӘиҰҒдҪ зңӢеҲ°дёҖеҖӢ Task жҳҜ CompletedпјҢйӮЈд»ЈиЎЁгҖҢиЈЎйқўзҡ„зЁӢејҸзўјзңҹзҡ„жІ’жңүзӮёжҺүгҖҚгҖӮйҖҷжҳҜ async/await иғҪиў«дҝЎд»»зҡ„ж №жң¬еҺҹеӣ пјҢTask зҡ„зӢҖж…ӢжҳҜз”ұгҖҢдҪ жҖҺйәјзөҗжқҹйӮЈж®өзЁӢејҸзўјгҖҚжұәе®ҡзҡ„!

рҹ”ө 1. д»ҖйәјйғҪдёҚијёе…Ҙ вҶ’ жӯЈеёёзөҗжқҹ

Console.ReadKey() жІ’жңүијёе…ҘпјҢdelayTask жҜҸж¬ЎйғҪе…Ҳе®ҢжҲҗ
5 ж¬ЎиҝҙеңҲеҫҢпјҢзӣҙжҺҘеӣһеӮі вҖңNo Input!вҖқпјҢд»ЈиЎЁд»»еӢҷжҳҜжӯЈеёёе®ҢжҲҗзҡ„пјҢзӢҖж…ӢжҳҜ Completed

рҹҹў 2. 5 з§’е…§ијёе…Ҙ A вҶ’ жӯЈеёёе®ҢжҲҗ

Console.ReadKey() ијёе…Ҙ A еҫҢпјҢinputTask е…Ҳе®ҢжҲҗгҖӮTask зӣҙжҺҘиө°еҲ° return вҖңOKвҖқгҖӮзӢҖж…ӢжҳҜ CompletedгҖӮ

рҹ”ҙ 3. 5 з§’е…§ијёе…Ҙ B вҶ’ зҷјз”ҹдҫӢеӨ–

B жңғеҹ·иЎҢ throw new ApplicationExceptionгҖӮйҖҷжңғи®“ Task и®ҠжҲҗ Faulted зӢҖж…ӢпјҲиЈЎйқўжңү ExceptionпјүгҖӮйҖІе…Ҙ catch (Exception) еҚҖеЎҠпјҢеҚ°еҮәйҢҜиӘӨиЁҠжҒҜгҖӮ

рҹҹЎ 4. 5 з§’е…§ијёе…Ҙ C вҶ’ еҸ–ж¶Ҳ

ијёе…Ҙ C жҷӮпјҢе‘јеҸ« cts.Cancel() дёҰдё” ThrowIfCancellationRequested()гҖӮйҖҷжңғжӢӢеҮә OperationCanceledExceptionгҖӮTask и®ҠжҲҗ Canceled зӢҖж…ӢгҖӮйҖІе…Ҙ catch (OperationCanceledException)
