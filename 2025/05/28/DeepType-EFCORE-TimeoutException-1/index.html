EF Core - TimeoutException (Part 1)

Created2025-05-28|Updated2026-03-08|иіҮж–ҷз–Ҷз•Ңзҡ„иҲӘең–
|Post Views:

жҹҗеӨ©ж—©дёҠпјҢзӣЈжҺ§зі»зөұи·іеҮәеҘҪе№ҫеҖӢ TimeoutExceptionпјҢжҲ‘еҖ‘е°Ҹзө„зҫӨзө„иЈЎеҸӘиҒҪеҲ°дёҖеҸҘпјҡгҖҢе•ҠпјҢжҮүи©ІжҳҜз¶Іи·Ҝе•ҸйЎҢеҗ§пјҢеҶҚи·‘дёҖж¬Ўе°ұеҘҪдәҶгҖӮгҖҚ

йҖҷеҸҘи©ұи®“жҲ‘иғҢи„ҠдёҖж¶јгҖӮ

TimeoutException дёҰдёҚжҳҜиіҮж–ҷеә«еңЁиҖҚи„ҫж°ЈпјҢе®ғжҳҜеЈ“еҠӣеӨӘеӨ§еңЁеҗ¶е–ҠгҖӮйҖҷзҜҮж–Үз« пјҢиҰҒдҫҶеҘҪеҘҪжўізҗҶйҖҷдәӣ вҖңеҗ¶е–ҠвҖқ иғҢеҫҢзҡ„зңҹжӯЈеҺҹеӣ 

жңүжҷӮеҖҷдҪ еҸӘжҳҜй–ӢеҖӢеҫҢеҸ°е ұиЎЁпјҢеҚ»й»ҳй»ҳжҹҘдәҶе№ҫеҚҒиҗ¬зӯҶиіҮж–ҷгҖӮд№…иҖҢд№…д№ӢпјҢдјәжңҚеҷЁе°ұи·ҹжіЎйәөдёҖжЁЈвҖ”вҖ”жіЎйҒҺй ӯпјҢиҶЁи„№зҲӣжҺү


1пёҸвғЈ зңҹзҡ„йңҖиҰҒе…ЁйғЁиіҮж–ҷе—Һпјҹ

е•ҶжҘӯжғ…еўғпјҡе ұиЎЁй–ӢзҷјеҲқжңҹзӮәжұӮз°ЎдҫҝпјҢеҸҜиғҪжңғзҝ’ж…ЈгҖҢе…Ёж’ҲгҖҚиіҮж–ҷдҫҶе‘ҲзҸҫпјҢдҪҶеҜҰйҡӣдҪҝз”ЁиҖ…еҸӘжңғзңӢжңҖиҝ‘зҡ„иЁҳйҢ„гҖӮ

2пёҸвғЈ дҪҝз”ЁжҠ•еҪұпјҲSelectпјүеҸӘжҠ“еҝ…иҰҒж¬„дҪҚ

е•ҶжҘӯжғ…еўғпјҡз•«йқўеғ…йЎҜзӨәдҪҝз”ЁиҖ…еҗҚзЁұиҲҮиЁ»еҶҠжҷӮй–“пјҢдҪҶеҚ»жҠҠеҖӢдәәз°Ўжӯ·гҖҒең°еқҖзӯүеӨ§еһӢж¬„дҪҚд№ҹж’ҲеҮәгҖӮ

3пёҸвғЈ дҪҝз”ЁзІҫжә–зҡ„жҹҘи©ўжўқд»¶пјҲWHEREпјү

е•ҶжҘӯжғ…еўғпјҡиЁӮе–®жҹҘи©ўжҸҗдҫӣдәҶеӨҡеҖӢжўқд»¶пјҲеҰӮзӢҖж…ӢгҖҒд»ҳж¬ҫж–№ејҸпјүпјҢеҚ»еҸӘдҪҝз”ЁжЁЎзіҠжҹҘи©ўйҖ жҲҗж•ҲиғҪдҪҺиҗҪгҖӮ



4пёҸвғЈ еҲҶжү№иҷ•зҗҶпјҲеҲҶй ҒжҹҘи©ў Pagingпјү

е•ҶжҘӯжғ…еўғпјҡеҫҢеҸ°з®ЎзҗҶзі»зөұзҡ„дҪҝз”ЁиҖ…жё…е–®пјҢеҲҶй ҒйЎҜзӨәпјҢжҜҸй Ғ 100 зӯҶ



5пёҸвғЈ дҪҝз”ЁжёёжЁҷејҸеҲҶй ҒпјҲKeyset PaginationпјүеҸ–д»Ј Offset

е•ҶжҘӯжғ…еўғпјҡз”ўе“ҒеҲ—иЎЁй Ғзҝ»еҲ°з¬¬ 100 й ҒпјҢдҪҝз”Ё Skip() ж•ҲиғҪи®Ҡе·®гҖӮеҸҜж”№д»ҘгҖҢжңҖеҫҢдёҖзӯҶ IDгҖҚдҪңзӮәжёёжЁҷгҖӮ

6пёҸвғЈ жҳҜеҗҰи©ІиҖғж…®иіҮж–ҷеҲҶеҚҖпјҲPartitionпјүжҲ–жӯёжӘ”

е•ҶжҘӯжғ…еўғпјҡдәӨжҳ“зҙҖйҢ„и¶…йҒҺдә”е№ҙпјҢе·ІдёҚеҶҚеёёз”ЁпјҢд»Қе…ЁйғЁж”ҫеңЁеҗҢдёҖејөиЎЁдёӯгҖӮ

е»әиӯ°й•·жңҹиіҮж–ҷе®ҡжңҹжӯёжӘ”гҖҒеҲҶиЎЁжҲ–дҪҝз”ЁиіҮж–ҷеҲҶеҚҖж©ҹеҲ¶

жҲ‘еҖ‘йғҪзҹҘйҒ“еңЁеёёиў«жҹҘи©ўжҲ–йҒҺжҝҫзҡ„ж¬„дҪҚдёҠе»әз«Ӣзҙўеј•еҸҜд»ҘеҠ йҖҹжҹҘи©ўгҖӮдҪҶеҜҰеӢҷдёҠ

гҖҢWHERE жҳҺжҳҺе‘ҪдёӯдәҶзҙўеј•пјҢзӮәд»ҖйәјжҹҘи©ўйӮ„жҳҜж…ўпјҹзҙўеј•дёҚжҳҜе°ұжҳҜзӮәдәҶеҠ йҖҹе—ҺпјҹгҖҚ

йҖҷиғҢеҫҢзҡ„й—ңйҚөжҳҜпјҢжҹҘеҲ°дәҶдҪҚзҪ®пјҢдёҚд»ЈиЎЁиіҮж–ҷе·Із¶“еңЁжүӢдёҠгҖӮйӮ„еҫ—зңӢдҪ жҹҘзҡ„ж¬„дҪҚжҳҜдёҚжҳҜд№ҹеҢ…еҗ«еңЁзҙўеј•дёӯгҖӮ

рҹ“ҡ жғіеғҸжҜ”е–»пјҡдҪ еңЁең–жӣёйӨЁжүҫдёҖжң¬жӣё

  1. дҪ з”Ёзӣ®йҢ„еҚЎпјҲзҙўеј•пјүжүҫеҲ°жӣёзҡ„дҪҚзҪ®пјҲWHERE е‘Ҫдёӯзҙўеј•пјү
  2. дҪ иө°еҲ°йӮЈеҖӢжӣёжһ¶пјҢжүҫеҮәйӮЈжң¬жӣё
  3. дҪҶйҖҷжң¬жӣёеҫҲеҺҡпјҢдҪ йӮ„жҳҜеҫ—жҠҠж•ҙжң¬жӣёзҝ»й–ӢдҫҶжүҫйӮЈдёҖй ҒпјҲSELECT зҡ„ж¬„дҪҚдёҚеңЁзҙўеј•иЈЎпјү

рҹ‘ү еҰӮжһңдҪ еҸӘжғізңӢжӣёзҡ„жҺЁи–ҰеәҸпјҢеӣ зӮәжІ’жңүзҙўеј•(дёҚзҹҘйҒ“дёҖзҸӯдҫҶиӘӘжңғеңЁе“Ә)йӮ„еҫ—жҠҠж•ҙжң¬жӣёжҹҘй–ұдёҖйҒҚпјҢеӣ жӯӨжҹҘеҫ—еҶҚеҝ«д№ҹеҸӘжҳҜгҖҢе®ҡдҪҚеҝ«гҖҚпјҢиіҮж–ҷйӮ„жҳҜиҰҒзҝ»еҚҠеӨ©гҖӮ

рҹ§  зңҹеҜҰиіҮж–ҷеә«дёӯзҷјз”ҹдәҶд»Җйәјпјҹ

  • Email жңүе»әз«Ӣзҙўеј• вң…
  • Address жІ’жңүеңЁзҙўеј•е…§ вқҢ
  1. Email е‘Ҫдёӯзҙўеј• вҶ’ еҝ«йҖҹе®ҡдҪҚи©ІзӯҶиіҮж–ҷеңЁдё»иЎЁзҡ„ RowId
  2. иіҮж–ҷеә«ж №ж“ҡзҙўеј• Row Pointer вҶ’ еӣһдё»иіҮж–ҷиЎЁж’ҲеҮә Address
  3. йҖҷзЁ®еӢ•дҪңзЁұзӮә Key Lookup / Bookmark Lookup

еҰӮжһңдҪ жҹҘи©ўеҫҲеӨҡзӯҶиіҮж–ҷпјҢйҖҷеҖӢеӣһдё»иЎЁзҡ„еӢ•дҪңе°ұжңғйҮҚиӨҮеҫҲеӨҡж¬ЎпјҢйҖ жҲҗеӨ§йҮҸзЈҒзўҹ I/OпјҢжҹҘи©ўж•ҲиғҪеҸҚиҖҢи®Ҡе·®гҖӮ

еңЁеёёжҹҘи©ўжҲ–йҒҺжҝҫзҡ„ж¬„дҪҚдёҠеҠ зҙўеј•гҖӮ

жҹҘи©ўиЎҢзӮә жҸҸиҝ° ж•ҲиғҪеҪұйҹҝ
е‘Ҫдёӯзҙўеј•дҪҶ SELECT ж¬„дҪҚдёҚеңЁзҙўеј•дёӯ еҝ«йҖҹе®ҡдҪҚпјҢдҪҶд»ҚйңҖеӣһдё»иЎЁ дёӯзӯүпјҢжңү I/O иІ ж“”
е®Ңе…ЁжІ’е‘Ҫдёӯзҙўеј• е…ЁиЎЁжҺғжҸҸ ж…ўпјҢиіҮжәҗж¶ҲиҖ—й«ҳ
дҪҝз”Ё Covering Index жүҖжңүиіҮж–ҷйғҪеңЁзҙўеј•дёӯеҸ–еҫ— жңҖеҝ«пјҢз„ЎйңҖеӣһдё»иЎЁ

зӯ”жЎҲжҳҜжңғпјҢй ҶеәҸеҪұйҹҝеҫҲеӨ§гҖӮ

еңЁе»әз«ӢиӨҮеҗҲзҙўеј•жҷӮпјҢзҙўеј•зҡ„й ҶеәҸжұәе®ҡдәҶе“ӘдәӣжҹҘи©ўеҸҜд»Ҙиў«жңүж•ҲеҲ©з”ЁгҖӮ

вң… д»ҘдёӢжҹҘи©ўиғҪжңүж•ҲдҪҝз”Ёи©Ізҙўеј•пјҡ

вқҢ дҪҶдёӢйқўзҡ„жҹҘи©ўе°ұз„Ўжі•дҪҝз”Ёзҙўеј•пјҡ

зҙўеј•еҸӘиғҪеҫһз¬¬дёҖеҖӢж¬„дҪҚй–Ӣе§ӢйҖЈзәҢдҪҝз”ЁпјҢдёҚиғҪи·іж¬„дҪҚ

JOIN жң¬иіӘжҳҜд»Җйәјпјҹ

йҖҷеҸҘи©ұзҡ„ж„ҸжҖқжҳҜпјҡжҜҸзӯҶиЁӮе–®еҺ»жүҫе°ҚжҮүзҡ„йЎ§е®ўпјҢзө„жҲҗж–°иіҮж–ҷиЎЁгҖӮжғіеғҸдёҖдёӢ Orders 1000 зӯҶгҖҒCustomers 100 зӯҶгҖҒRegions 10 зӯҶ

жІ’жңүйҷҗеҲ¶жўқд»¶жҲ–зҙўеј•пјҢJOIN еҮәдҫҶзҡ„дёӯй–“иЎЁеҸҜиғҪжҳҜ 1000 Г— 100 Г— 10 = зҷҫиҗ¬зҙҡиіҮж–ҷгҖӮ

  • жІ’жңүзҙўеј• вҶ’ Hash JoinпјҲе»әз«Ӣ Hash TableпјҢеҶҚжҜ”е°Қпјү
  • LEFT JOIN вҶ’ иіҮж–ҷдёҚдёҖе®ҡе‘ҪдёӯпјҢжӣҙж…ў
  • еӨҡеұӨ JOIN вҶ’ дёӯй–“иЎЁзҲҶйҮҸпјҢиЁҳжҶ¶й«”зӮёиЈӮ

иіҮж–ҷеә«жҖҺйәјеҹ·иЎҢ JOINпјҹ

  • еҫһдё»иЎЁпјҲеҒҮиЁӯжҳҜ OrdersпјүжҺғеҮәдёҖзӯҶ
  • ж №ж“ҡ CustomerId еҺ» Customers иЎЁдёӯ жүҫеҮәе°ҚжҮүзҡ„ Id
  • жҠҠе…©зӯҶиіҮж–ҷзө„еҗҲиө·дҫҶ

йҖҷеҖӢгҖҢеҺ»жүҫе°ҚжҮүиіҮж–ҷгҖҚзҡ„еӢ•дҪңпјҢиғҪдёҚиғҪз”Ёзҙўеј•дҫҶеҠ йҖҹпјҹ

дҪҝз”Ёжңүзҙўеј•зҡ„ж¬„дҪҚйҖІиЎҢй…Қе°Қ

  • Customers.Id жҳҜдё»йҚөпјҢиҮӘеӢ•жңүзҙўеј•
  • иіҮж–ҷеә«еҸҜд»ҘйҖҸйҒҺзҙўеј•гҖҢеҝ«йҖҹе®ҡдҪҚгҖҚз¬ҰеҗҲжўқд»¶зҡ„йӮЈзӯҶе®ўжҲ¶иіҮж–ҷ
  • JOIN ж•ҲиғҪеҘҪпјҢйҖҡеёёжңғдҪҝз”Ё Nested Loop Join + Index Seek

жңӘдҪҝз”Ёжңүзҙўеј•зҡ„ж¬„дҪҚдҫҶй…Қе°Қ

  • еҰӮжһңдҪ  JOIN зҡ„ж¬„дҪҚдёҚжҳҜзҙўеј•пјҢиіҮж–ҷеә«еҝ…й Ҳе°Қ Customers ж•ҙејөиЎЁйҖҗзӯҶжҺғжҸҸ
  • еҠ дёҠ JOIN жҳҜеӨҡе°ҚеӨҡеҢ№й…ҚпјҢиіҮж–ҷйҮҸд№ҳеҖҚеўһеҠ 
  • жңҖзөӮе°ҺиҮҙе…ЁиЎЁжҺғжҸҸпјҲTable ScanпјүжҲ–й«ҳжҲҗжң¬зҡ„ Hash Join

Hash Join жҳҜдёҖзЁ®гҖҢе…Ҳе»әиЎЁпјҢеҶҚжҜ”е°ҚгҖҚзҡ„ж–№ејҸгҖӮиіҮж–ҷеә«жңғе…ҲжҠҠе…¶дёӯдёҖејөиЎЁж”ҫйҖІиЁҳжҶ¶й«”дёӯпјҢз”Ёжҹҗж¬„дҪҚе»әз«ӢHash TableпјҢеҶҚз”ЁеҸҰдёҖејөиЎЁдҫҶжҜ”е°ҚгҖӮ

жӢҶжҹҘи©ўпјҢеҶҚзө„иө·дҫҶ

е°ҮиӨҮйӣң JOIN жӢҶжҲҗж•ёеҖӢе°ҸжҹҘи©ўпјҢйҖҗжӯҘеҸ–еҫ—еҝ…иҰҒиіҮж–ҷ вҶ’ жҜҸж¬ЎйғҪиғҪе‘Ҫдёӯзҙўеј•гҖҒиіҮж–ҷйҮҸе°ҸгҖӮжңҖеҫҢеңЁжҮүз”ЁеұӨз”ЁиЁҳжҶ¶й«”жҲ– LINQ еҗҲдҪөгҖӮ

TimeoutException дёҚи©Іиў«еҝҪз•ҘпјҢе°ұеғҸ GPS жҢҒзәҢиҪүеңҲеңҲпјҢиғҢеҫҢеҸҜиғҪжҳҜиіҮж–ҷж’ҲеӨӘеӨҡгҖҒJOIN йҒҺйҮҚгҖҒзҙўеј•иЁӯиЁҲйҢҜиӘӨжҲ–жўқд»¶дёҚзІҫжә–гҖӮ
и®“жҲ‘еҖ‘еҲҘеҶҚеҝҪиҰ–йҖҷдәӣиЁҠиҷҹпјҢеҘҪеҘҪеӣһжҮүиіҮж–ҷеә«зҡ„е‘јж•‘иҒІгҖӮд№ҹиЁұе®ғдёҚжҳҜдёҚжғіеӣһдҪ пјҢеҸӘжҳҜйңҖиҰҒдёҖй»һз©әж°Ј
