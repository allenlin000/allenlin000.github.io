NotSupportedException:

描述: 當試圖使用資料庫不支援的功能時引發，或者當特定操作不適用於當前操作的提供者時引發。

例子: 嘗試使用某些特定於 SQL Server 或 SQLite 的功能，但資料庫供應者不支援這些功能。
